# Trabajo Práctico - Gestión Colaborativa, Control de Versiones y Organización Empresarial (Git, GitHub y Jira)

## Descripción general
Este proyecto fue desarrollado como parte de la materia Organización Empresarial de la Tecnicatura Universitaria en Programación a Distancia (UTN).
El objetivo principal del trabajo es aplicar herramientas de gestión colaborativa y control de versiones utilizando Git, GitHub, Google Colab y Jira,
simulando el flujo de trabajo de una célula de desarrollo de software.
El proyecto consiste en el análisis de un conjunto de datos de ventas comerciales, permitiendo obtener información relevante sobre artículos vendidos, cantidades,
ingresos y comportamiento general de las ventas.

## Herramientas utilizadas
- Git
- GitHub
- Google Colab
- Jira Software
- Python
- CSV como fuente de datos
